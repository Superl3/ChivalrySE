super.StartRound()
다음에 밸런스를 실행한다.


합리적인 해결방안
일단 FFA에선 밸런싱 설정 X
갑자기 바꾸는것은 금지하고
최대한 덜 바꾸는 방법을 지향한다

1. FFA 인식 후 밸런싱 설정 생략
2. 무조건 합리적인게 않더라도 한명 나갔을 경우 한명만 스왑해서 최적의 해를 찾는다.
 + 한명만 뽑는데 팀원 수가 두명 이상 차이날 경우 아예 리밸런싱 시켜줌
3. 데스 베이스드 오토밸런싱을 존중해서 팀만 바꿔준다.

인원이 같을 경우 -> 랭크 계산해서 합리적일 경우 ㄱㄱ
인원이 한명 차이날 경우 -> 한명을 바꾸거나 아니면 일방적으로 옮기거나
인원이 두명 차이날 경우 -> 해당 알고리즘 한번 더 수행

-> 언제 수행되는가
-> 누군가가 나갔을 경우 죽었을때 수행하도록 명령
